week 4: A Brief History of Programming 
Display and Positioning 
local repo: A_Brief_History_Of_Programming

A browser will render the elements of an HTML document that has no CSS from left to right, top to bottom, in the same order as they exist in the document. This is called the flow of elements in HTML.  In addition to the properties that it provides to style HTML elements, CSS includes properties that change how a browser positions elements.   In this lesson, you will learn three properties for adjusting the position of HTML elements in the browser: POSITION, DISPLAY, Z-INDEX.
